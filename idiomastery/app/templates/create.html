Submission Guidelines

1. Idiom Format

  • Use proper spelling, grammar, and punctuation.
  • Avoid slang, profanity, or phrases with offensive connotations.

2. Meaning

  • Provide a clear and concise explanation of what the idiom means.

3. Origin

  • If known, briefly describe the origin (historical, cultural, or linguistic).
  • Cite a source if possible.

4. Equivalent Idioms

  • Add idioms from other languages or regions that express a similar idea.
  • Clearly label the language or region if adding non-English equivalents.

5. Example Usage

  • Include a sentence showing how the idiom is used naturally in context.

6. Translations

  • Add translations only if you are confident in their accuracy.
  • Choose idiomatic rather than literal translations where possible.
